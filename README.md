# Predictive-Analysis-of-Sustainable-fashion-using-ML

## Overview

This project focuses on analyzing and predicting consumer behavior toward sustainable fashion using machine learning techniques. The objective is to identify patterns in consumer purchasing decisions and understand the factors influencing sustainable fashion adoption.

This work is based on a **published research study** that applies machine learning and data analysis techniques to study consumer attitudes toward sustainable fashion.



## Research Objective

The primary objectives of this project are:

* Analyze consumer behavior and preferences regarding sustainable fashion.
* Identify key factors influencing sustainable purchasing decisions.
* Apply machine learning models to predict consumer behavior.
* Interpret model predictions using explainable AI techniques.



## Machine Learning Algorithms Used

The following machine learning models were implemented and evaluated:

* **Logistic Regression**
* **Decision Tree Classifier**
* **Random Forest Classifier**
* **XGBoost Classifier**

To address class imbalance in the dataset:

* **SMOTE (Synthetic Minority Over-sampling Technique)** was applied.



## Dimensionality Reduction & Visualization

To better understand the structure of the dataset and visualize feature relationships:

* **Principal Component Analysis (PCA)** was used for dimensionality reduction and data visualization.



## Model Interpretability

To improve transparency and understand feature importance:

* **SHAP (SHapley Additive exPlanations)** was applied to explain model predictions and identify the most influential features affecting consumer behavior toward sustainable fashion.



## Project Workflow

The project workflow includes the following steps:

1. Data Collection and Dataset Understanding
2. Data Cleaning and Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Encoding and Transformation
5. Handling Class Imbalance using SMOTE
6. Dimensionality Reduction using PCA
7. Model Training using Multiple Machine Learning Algorithms
8. Model Evaluation and Comparison
9. Model Interpretability using SHAP Analysis
10. Visualization and Result Interpretation

## Results

The machine learning models were used to analyze consumer behavior patterns toward sustainable fashion. The models demonstrated strong predictive capability and provided insights into the factors influencing consumer decisions.

Model performance was evaluated using:

* Accuracy
* Precision
* Recall
* F1-score

Additionally, **SHAP analysis** provided interpretability by highlighting the most influential features contributing to sustainable fashion adoption.
